# 🐰 万兔福 - 一分钟了解

## 🎯 这是什么？

**万兔福**是一个基于OpenClaw框架的智能股票分析系统，采用"一主四辅"分布式架构。

## 📖 名字含义

- **万** = 万物皆可分析（全面性）
- **兔** = 快速如兔（敏捷性，谐音"two"代表一主四辅）
- **福** = 福运相伴（收益性）

**口号**: 万物皆可分析，快速如兔，福运相伴

## 🏗️ 系统架构

```
          🦸 万兔福总指挥 (主Agent)
                  │
          ┌───────┴───────┐
          │               │
    🐇 万兔之眼     🐇 万兔之脑
    (数据员)        (策略员)
          │               │
          └───────┬───────┘
                  │
          ┌───────┴───────┐
          │               │
    🐇 万兔之镜     🐇 万兔之盾
    (回测员)        (风控员)
```

## ✨ 核心功能

1. **🐇 万兔之眼**: 实时数据采集和监控
2. **🐇 万兔之脑**: 智能策略分析和决策
3. **🐇 万兔之镜**: 历史回测和验证
4. **🐇 万兔之盾**: 风险控制和保护
5. **🦸 万兔福总指挥**: 系统协调和报告生成

## 🚀 快速开始

```bash
# 1. 获取万兔福
git clone https://github.com/qhdhao13/1to4-stock.git
cd 1to4-stock

# 2. 安装依赖
pip install aiohttp akshare pandas

# 3. 启动系统
python3 start_all.py

# 4. 测试系统
python3 quick_test.py
```

## 📊 技术特点

- **分布式架构**: 5个独立Agent协同工作
- **实时分析**: 秒级响应，快速决策
- **全面覆盖**: 基本面+技术面+风险面
